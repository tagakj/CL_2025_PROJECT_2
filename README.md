# CL_2025_PROJECT_2

Тема: Классификация обращений пациентов.

Цель: Проверить гипотезу, что векторизация текстов обращений с добавлением биграмм улучшит качество классификации.

Материалы: 1708 обращений пациентов на сайте https://mamadeti.ru/

Этапы:
1.	парсинг сайта, предобработка текстов, лемматизация
2.	визуализация данных (с использованием TSNE)
3.	сравнение моделей классификации

Выводы: Добавление биграмм в признаки текстов помогает улучшить качество классификации.
